# Руководство администратора Plakor Divisiones

## Содержание
1. Вход в админ-панель
2. Управление проектами
3. Обработка заявок
4. Управление контентом
5. SEO настройки

## 1. Вход в админ-панель
- Перейдите по адресу: `https://admin.plakordivisiones.es`
- Введите ваши учетные данные
- При первом входе смените пароль

## 2. Управление проектами
### Добавление нового проекта
1. Перейдите в раздел "Projects"
2. Нажмите "Create new entry"
3. Заполните поля:
   - Title: название проекта
   - Description: описание работ
   - Category: категория (выберите из списка)
   - Image: основное изображение (рекомендуемый размер 1200x800px)
   - Gallery: дополнительные фото
   - Slug: URL проекта (генерируется автоматически)

### Редактирование проектов
- Используйте фильтры для поиска
- Нажмите на проект для редактирования
- Не забудьте сохранить изменения

## 3. Обработка заявок
### Просмотр новых заявок
1. Перейдите в раздел "Solicitudes"
2. Новые заявки отмечены статусом "pendiente"
3. Для каждой заявки доступна информация:
   - Nombre: имя клиента
   - Teléfono: контактный телефон
   - Tipo: тип работ
   - Fecha: дата заявки

### Обработка заявок
1. Откройте заявку
2. Измените статус:
   - pendiente: новая
   - en_proceso: в обработке
   - completada: завершена
   - cancelada: отменена
3. Добавьте комментарий при необходимости

## 4. Управление контентом
### Редактирование текстов
1. Перейдите в раздел "Content"
2. Выберите нужный раздел
3. Используйте редактор для форматирования

### Управление изображениями
- Загружайте изображения в формате JPG/PNG
- Оптимизируйте размер (макс. 2MB)
- Используйте говорящие названия файлов

## 5. SEO настройки
### Метатеги
1. Для каждой страницы настройте:
   - Title: 50-60 символов
   - Description: 150-160 символов
   - Keywords: основные ключевые слова

### Оптимизация изображений
- Добавляйте alt-текст
- Используйте оптимизированные изображения
- Следите за размером файлов

## Безопасность
- Регулярно меняйте пароль
- Не передавайте учетные данные
- Используйте безопасное соединение
